Amazon Delivery Time Analysis

This data science exploration investigates how various factors - including weather conditions, geographical location (urban vs. rural), and traffic congestion - impact Amazon delivery times. Using a comprehensive dataset from Kaggle containing over 43,000 delivery records, I developed predictive models to forecast delivery times and identify key influencing factors.

View Presentation

Overview

The analysis revealed several interesting insights: groceries have significantly shorter delivery times compared to other product categories; there's a positive correlation between driver age and delivery time; higher agent ratings correspond with faster deliveries; and weather conditions like cloudy or foggy days actually facilitate quicker deliveries. The Random Forest model outperformed Linear Regression in predictive accuracy, achieving an R-squared score of 0.812.

This analysis demonstrates how data-driven insights can enhance operational efficiency, route optimization, and ultimately customer satisfaction in logistics operations.

Technologies Used

  • Python: Primary programming language for data analysis and modeling
  • Pandas & NumPy: Data manipulation and preprocessing
  • Matplotlib & Seaborn: Data visualization
  • Scikit-learn: Machine learning model implementation (Random Forest and Linear Regression)
  • Haversine Library: Distance calculation between geographic coordinates
  • Jupyter Notebooks: Development environment for code and analysis
  • Git/GitHub: Version control and project management
